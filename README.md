## 📌 Overview
**Maskan** is a modern, responsive e-commerce storefront built to practice core frontend development skills. It focuses on clean UI and functional interactivity using vanilla technologies.

## ✨ Key Features
* **Shopping Cart:** Add/remove items and manage quantities.
* **Wishlist:** Save favorite products for later.
* **Product Details:** Dynamic viewing of product information.
* **Fully Responsive:** Optimized for all screen sizes (Mobile, Tablet, Desktop).

## 🛠️ Tech Stack
* **HTML5**
* **Tailwind CSS** (Utility-first styling)
* **JavaScript** (Core logic and interactivity)

## 🎯 Purpose
This project was developed as a hands-on exercise to master **JavaScript** logic and efficient layout building with **Tailwind CSS**.
